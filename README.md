🌸 SheCan Foundation – Tech Internship Application Portal
A full-stack web application built to collect and manage internship applications for SheCan Foundation.
This project includes a user-friendly frontend, backend APIs, authentication system, and an admin dashboard.


🚀 Features

👩‍💻 User Side
* User Signup & Login
* Secure form submission
* Internship application form
* Validation for required fields
* Success message after submission

🔐 Admin Side
* Admin login system
* View all submitted applications
* Protected admin dashboard
* Logout functionality

⚙️ Backend Features
* REST APIs using Express.js
* SQLite database integration
* Authentication (role-based: user/admin)
* Data storage for applications and users


🛠️ Tech Stack

Frontend:
* HTML
* CSS
* JavaScript

Backend:
* Node.js
* Express.js

Database:
* SQLite

⚙️ Installation & Setup

1️⃣ Clone the repository
git clone https://github.com/your-username/shecan.git
cd shecan

2️⃣ Install dependencies
cd server
npm install

3️⃣ Run the server
node server.js (1st terminal inside server)

4️⃣ Open in browser
open index.html (2nd terminal inside SheCan)


🧪 How It Works
1. User creates an account or logs in
2. User fills the internship form
3. Data is sent to backend API
4. Stored in SQLite database
5. Admin logs in to view all applications
